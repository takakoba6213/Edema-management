📋 初期評価チェックリスト
  • 片側性に見えても両側性の初期、両側性でも左右差があれば片側要素の合併を考慮する
  • 急性発症の目安は3日以内。慢性経過では周囲からの指摘で気づくことが多い
  • 非圧痕性浮腫（non-pitting edema）の鑑別：脂肪浮腫・血管性浮腫・甲状腺機能低下症・皮膚筋炎・強皮症・パルボウイルスB19感染・黄色爪症候群・リンパ浮腫
  • 50歳以上の慢性・両側性 → 慢性静脈不全（女性に約2倍多い）をまず考える
  • 若年者 → 睡眠時無呼吸症候群（SAS）・月経関連・妊娠・パルボウイルスB19・好酸球性血管性浮腫（20代女性に多い）・脂肪浮腫（女性に多い）
  • 小児 → ネフローゼ症候群・溶連菌感染後糸球体腎炎（PSGN）・アレルギー疾患
  • 高齢者の器質的疾患 → RS3PE症候群・リウマチ性多発筋痛症（PMR）・上大静脈症候群（SVC）・Trousseau症候群・麻痺側の浮腫
  • 浮腫以外の症状がない外来患者 → 慢性静脈不全か薬剤性をまず考える
📊 浮腫を呈する疾患の頻度と臨床的重要度

頻度・重要度とも高い：心不全・腎疾患／頻度中等度：肝疾患／それ以下：栄養障害性・薬剤性・特発性

2. 浮腫の病態生理（4つの機序）▼
機序主な原因疾患
① 膠質浸透圧低下 低アルブミン血症（血清Alb<2.0 g/dL）：ネフローゼ症候群・肝硬変・低栄養・蛋白漏出性胃腸症
② 静水圧上昇 心不全・腎不全・肝硬変・物理的圧迫、細動脈平滑筋収縮不全（高齢者・糖尿病・Ca拮抗薬・Parkinson病・抗コリン薬）、静脈閉塞・静脈弁不全
③ 血管透過性亢進 ヒスタミン・ブラジキニン・サイトカイン・VEGFの関与（アレルギー・炎症・血管炎）
④ リンパ還流障害 腫瘍による圧迫・リンパ節郭清後・フィラリア症・放射線治療歴
🔬 病態生理チェック
  • 間質への物質沈着が原因となる疾患：バセドウ病・甲状腺機能低下症・浮腫性硬化症
  • 甲状腺機能低下症：TSH受容体を介した刺激で線維芽細胞がグリコサミノグリカンを過剰産生 → ゲル化し、圧迫しても凹まない（粘液水腫）
  • エストロゲン・バソプレシン・アルドステロンはいずれも浮腫を促進する
  • NSAIDsはナトリウム貯留を介して浮腫を生じる
  • Ca拮抗薬（CCB）は約50%で下腿浮腫を生じる
  • 細動脈は3層構造で平滑筋があり、基本的に血液は漏出しない
  • 毛細血管は内皮細胞・ペリサイト・基底膜のみで構成され、組織により透過性が異なる（脳では密）
  • 血漿成分は主に毛細血管から漏出する
  • 間質液は1日3 Lまではリンパ管により回収されうる
  • DVTなどで細静脈圧が上昇すると、間質から血管内への体液回収が低下する
  • SAS・月経・特発性・パルボウイルスB19・アミロイドーシスも浮腫の原因となる
3. 問診▼
問診の軸：全身性／局所性 ／ 両側性／片側性 ／ 発症様式（日内変動含む）／ 随伴症状（体重増加の有無）／ 薬剤歴 ／ 生活様式（体位・職業・塩分摂取・飲水量・尿量）／ review of systems
⏱ Onset（発症様式）
  • 急性発症（3日以内）：DVT・Baker嚢胞破裂・蜂窩織炎・外傷・血管性浮腫・全身疾患の急性増悪
  • 慢性発症：慢性静脈不全・リンパ浮腫・慢性全身性疾患
🔄 反復性
  • 血管性浮腫の反復（トリガーなし・無痛性）
  • リンパ浮腫に続発する反復性蜂窩織炎
📍 部位
  • 片足の浮腫でも全身性浮腫の初発症状である可能性を常に念頭に置く
  • 左右差がある場合は血流障害・麻痺などの局所的要因が重なっている可能性を考える
  • 眼瞼浮腫 → ネフローゼ症候群・血管性浮腫を考える
📈 増悪・軽快因子
  • 夕方に増悪 → 静脈圧上昇機序（慢性静脈不全・心不全）を示唆
🔍 圧痕性の評価
  • Fast edema（10秒圧迫後40秒以内に戻る）→ 早期の低アルブミン血症を示唆 Saudi Med J 2016
  • 低アルブミン血症でも長期化するとSlow edemaに移行しうる
  • 圧痕性（pitting）の有無だけでは原因を絞れないことを理解する
  • 非圧痕性（non-pitting）：血管性浮腫・リンパ浮腫・甲状腺機能低下症・脂肪浮腫・パルボウイルスB19・強皮症・黄色爪症候群
🩺 基礎疾患別の特徴
  • 悪性腫瘍では複数機序が併存しうる：DVT・腫瘤による圧迫・放射線治療後・リンパ節転移・浸潤・低アルブミン血症
  • 糖尿病関連：糖尿病性腎症・心不全・末梢神経障害・薬剤性・神経因性膀胱
  • 黄色爪症候群（yellow nail syndrome）
  • 栄養失調型浮腫
  • アルコール多飲 → 栄養障害を介した浮腫

4. 身体診察▼
圧痕性浮腫の評価法：5 mmの深さで10秒間圧迫し、回復時間で判定
・Fast edema（2秒で圧痕、2〜3秒で復元）→ 低アルブミン血症
・Slow edema（10秒で圧痕、40秒経過しても復元しない）→ その他の原因
  • 全身性浮腫では血圧・脈拍・呼吸数・心肺聴打診・肝脾腫・腹水・頸静脈怒張を系統的に評価
  • 頸静脈圧：胸骨角から3 cm以上なら中心静脈圧上昇と判断
  • 下腿の色素沈着 → ヘモジデリン沈着を反映、慢性静脈不全を示唆
  • 皮下出血所見 → アミロイドーシスを示唆
  • 腹部の横断所見 → Budd-Chiari症候群（側副血行）を考慮
  • Stemmer徴候（第2趾をつまめない） → リンパ浮腫（感度92%・特異度57%） Plast Reconstr Surg GO 2019
  • 肝硬変 → 黄疸・クモ状血管腫・手掌紅斑・女性化乳房
  • 甲状腺機能低下症 → 顔面浮腫・眉毛外側脱落・皮膚乾燥・嗄声・低体温・徐脈
  • 麻痺側で浮腫が目立つ／臥床患者では仙骨部・背部に浮腫が現れる
  • 両側性でも解剖学的に左側優位となりうる（左総腸骨静脈の圧迫）

6. 胸水・腹水の鑑別▼
🫁 胸水：Light基準
Light基準（1項目以上を満たせば滲出性）：
・蛋白比（胸水/血清）> 0.5
・LDH比（胸水/血清）> 0.6
・胸水LDH > 血清LDH基準値上限の2/3
利尿薬使用中など偽滲出性の場合：胸水コレステロール> 60 mg/dL、血清−胸水Alb < 1.2 g/dLなら漏出性と再判断
A. 漏出性胸水
疾患参考所見
心不全利尿薬使用中は滲出性化することあり。NT-proBNP> 1,300 pg/mL（感度96%・特異度88%）、血清−胸水Alb> 1.2 g/dL
低Alb血症・ネフローゼ症候群通常両側性
肝性胸水（肝硬変）70%で右側のみ、多くは腹水を伴うが単独もあり。血清−胸水Alb> 1.1 g/dL
髄液胸水／尿胸水V-Pシャント合併症、尿管閉塞・外傷。胸水Cr/血清Cr> 1
無気肺胸腔内圧低下による

  • 胸水Ht/末梢血Ht > 0.5 → 血胸
  • 血性胸水 → 悪性腫瘍・外傷・肺塞栓

🔬 腹水：SAAGによる鑑別
SAAG ≧ 1.1 g/dLで門脈圧亢進（97%）：肝硬変・アルコール性肝炎・心不全・収縮性心膜炎・Budd-Chiari症候群・髄液腹水・尿腹水・甲状腺機能低下症
SAAG < 1.1 g/dL：癌性腹膜炎・細菌性腹膜炎・結核性腹膜炎・膵性腹水・腸閉塞・虚血・漿膜炎・胆汁性腹膜炎・膵炎・ネフローゼ症候群
  • 腹水を呈する疾患の頻度：肝硬変81%・癌10%・心不全3% JAMA 2008/2012
  • 腹水・食道静脈瘤・黄疸・肝性脳症のいずれかがあれば非代償性肝硬変
  • TG > 200 mg/dL → 乳糜胸腹水
  • 血性腹水 → 肝細胞癌
  • アミラーゼ腹水/血清比> 0.4 → 膵炎・消化管穿孔
  • 癌性腹膜炎の支持所見：コレステロール≧ 70 mg/dL（感度88%・特異度100%）、CEA ≧ 5.0、腹水LDH ≧ 200
  • 腹膜透析関連腹膜炎：WBC > 100/μL
📝 忘れがちな鑑別
  • 薬剤性・石綿肺・黄色爪症候群・trapped lung・尿胸／腹水・髄液胸水・アミロイドーシス・収縮性心膜炎も鑑別に入れる
7. 上肢浮腫▼
  • 若年者の急性両側上肢浮腫 → 好酸球性血管性浮腫、パルボウイルスB19。慢性であれば月経関連・妊娠・甲状腺疾患 Immunol Allergy Clin 2006
  • 中年 → 皮膚筋炎・強皮症も鑑別に加える
  • 高齢 → 関節リウマチも考慮
  • 運動負荷後の急性上肢浮腫 → 静脈損傷による血栓閉塞（Paget-Schroetter症候群）
  • ワクチン接種後の上肢浮腫 → 後天性血管性浮腫（リンパ腫・SLE・感染症・ワクチン関連）
  • 無症候性の脳血管障害後に浮腫が現れることがある
  • 上肢DVTを認めたら悪性腫瘍を疑う
  • 浮腫と思われる所見が、移動性関節炎・淋菌関節炎であることもある
  • CRPS（四肢の熱感・腫脹）は脳梗塞・心筋梗塞後に続発することがある
  • 女性の片側上肢浮腫 → 乳癌によるリンパ浮腫を必ず除外
  • RS3PE症候群 → 悪性腫瘍を必ず除外する
  • PICCカテーテル遠位では血流低下により浮腫の原因となる
  • 両側上腕のみの浮腫 → 脂肪浮腫（lipedema）

📝 下肢浮腫の追加チェック
  • Baker嚢胞の背景：変形性膝関節症（OA）、関節リウマチ、半月板断裂
  • Baker嚢胞破裂：NSAIDs投与で6〜12週で自然軽快 Cureus 2021
  • 糖尿病患者の突然の片側下肢腫脹 → 筋肉内血腫・糖尿病性筋梗塞（末梢血管障害と同様の機序）
  • しびれ＋浮腫 → ビタミンB1欠乏（脚気）を考える（脚気心から心不全・浮腫をきたす）
  • 発作性の浮腫が数週〜数ヶ月ごとに反復する → 好酸球性血管性浮腫
  • 若年女性の四肢末梢浮腫 → パルボウイルスB19だけでなく好酸球性血管性浮腫・月経関連浮腫も考慮 Immunol Allergy Clin 2006 Clin Rheumatol 2006
  • DVT後は後遺症として静脈瘤・色素沈着などの慢性静脈不全が生じうる
  • 特発性浮腫のThorn基準：朝夕の体重差1.4 kg以上、浮腫をきたす基礎疾患なし、精神的障害か情動不安定
  • 下剤乱用 → 持続的RAS活性化 → Na貯留・浮腫。下剤中止後も遷延することあり
  • 皮膚硬化を伴う浮腫 → 強皮症・好酸球性筋膜炎を考える
  • 激しい労作後の浮腫性硬化・動作時痛・皮膚の凹凸 → 好酸球値に関わらず好酸球性筋膜炎を考える Autoimmun Rev
  • ドパミンアゴニスト → 2年以上内服の19%に薬剤性浮腫
  • 好酸球性筋膜炎には血液悪性腫瘍の合併あり Curr Rheumatol Rep 2017
  • 筋膜炎の原因：感染症・自己免疫性・良性腫瘍
  • SLE・ANCA関連血管炎・PANでも筋膜炎が生じうる
  • 痛風も浮腫の原因となる。痛風発作は外傷がトリガーとなることがある
  • May-Thurner症候群（左総腸骨静脈が右総腸骨動脈に圧迫される）Lancet 2019
  • 筋肉内血腫・皮下出血が浮腫の原因となることがある
  • 運動後の激痛 → コンパートメント症候群

11. 治療▼
基本方針：原疾患治療＋塩分制限＋利尿薬の組み合わせ
例外：血管透過性亢進・リンパ浮腫・DVTでは利尿薬は逆に病態を悪化させる
💊 利尿薬の使い方
  • 浮腫＝間質液貯留であり、必ずしも血管内容量増加を意味しない
  • 腹腔内から血管内への体液移動は1日300〜500 mL程度と緩徐
  • 尿細管でのNa再吸収：近位尿細管60%、Henle上行脚30%、遠位尿細管9%
  • スピロノラクトンは集合管でのNa再吸収を阻害
  • 利尿薬は血流で腎臓へ運ばれ、尿細管腔に分泌され、輸送タンパクと結合して作用する
  • フロセミドの生物学的利用能（bioavailability）は平均50%だが10〜90%と個人差が大きい
  • 低アルブミン血症では利尿薬がタンパク結合できず効果が減弱
  • タンパク非結合型利尿薬は近位尿細管の陰イオントランスポーターで分泌され作用する
  • NSAIDs・プロベネシドはこのトランスポーターを阻害 → 利尿効果減弱
  • 腎機能低下時は輸送能力低下により利尿薬の効果も低下
  • ループ利尿薬は1日2回が望ましい。1日1回では反跳的にpost-diuretic NaCl retentionが起こりNa排泄が低下する
  • 慢性使用でブレーキ現象（RAS活性化、ネフロン再構築）が起こりループが効きにくくなる → サイアザイド系・ミネラルコルチコイド受容体拮抗薬（MRA）を併用することで対抗可能
  • ループ利尿薬で反応尿が少ない場合、2〜3回同量反復し、2時間後尿Na< 50 mmol/Lなら倍量化を検討
  • 1週間以上続いて効果不十分なら、サイアザイド系 → 2〜5時間後にループ利尿薬という順序投与
  • 低K血症が出現したらMRA追加（効果発現に1〜2日必要）
  • フロセミドは持続投与のほうが耳毒性が低いとされる
  • アルブミン＋フロセミド併用はAlb < 2 g/dLで検討可だが、有効性を否定する報告もある
🩹 その他の治療
  • 塩分制限：Na 6 g/日以内
  • リンパ浮腫の背景：悪性腫瘍治療（リンパ節郭清・術後照射・タキサン系抗癌剤）
  • リンパ浮腫の治療：用手的リンパドレナージ・弾性ストッキング・患側圧迫下での運動・スキンケア・自己管理指導
🧦 圧迫療法 Am Fam Physician 2022
  • 圧迫療法はほとんどの下肢浮腫の原因に有効（SORT推奨）
  • CVI：段階的弾性ストッキング（20〜40 mmHg）が第一選択。足関節で最も強く、上方に向かって圧が低下する
  • 弾性包帯・調節式圧迫装具・間欠的空気圧迫も選択肢
  • 開始前にABI（足関節上腕血圧比）を確認：末梢動脈疾患（PAD）があれば圧迫は禁忌
  • リンパ浮腫：まず非弾性圧迫包帯で浮腫を軽減 → 改善後に弾性ストッキング（40 mmHg以上）へ移行
🏥 疾患別マネジメント Am Fam Physician 2022
疾患治療のポイント
CVI圧迫療法＋体重減量・フレボトニクス（静脈賦活剤）。セイヨウトチノキ種子エキス（horse chestnut seed）は中等度のエビデンスで浮腫軽減。ルスクスエキスも疼痛・浮腫に有効。ペントキシフィリンは静脈性潰瘍に有効
リンパ浮腫圧迫療法＋用手的リンパドレナージ＋運動＋スキンケア。持続・進行例はリンパ外科医へ紹介（debulking/bypass術）。リンパ静脈バイパスで周径1.9〜2.4 cm減少の報告あり
DVT直接作用型経口抗凝固薬（DOAC：factor Xa阻害薬）が第一選択。最低3ヶ月継続。再発例・誘因なしの場合は長期投与を検討。血栓後症候群（PTS）は25〜50%に発症
心不全利尿薬抵抗性CHFではトルセミド（torsemide）がフロセミドより有効（入院・心血管死亡リスク減少：SORT A）。肝硬変による腹水にはスピロノラクトン:フロセミド = 5:2比で開始
薬剤性原因薬剤の中止・変更が原則。CCB → ACE阻害薬/ARBへの切替えまたは併用を検討
妊娠足の挙上、左側臥位（IVC圧迫の軽減）、弾性ストッキング。塩分制限も考慮
非全身性浮腫に利尿薬を使うエビデンスはない。全身性疾患以外の浮腫に安易に利尿薬を使用しない
12. ピットフォール▼
浮腫治療はオーバー治療になりがち！ 一つの病態で説明しようとしない。複数の原因が併存することが多い。
  • 複合病態への意識：常に複数原因の併存を想定する
  • うっ滞性皮膚炎＋蜂窩織炎の併存
  • ネフローゼ症候群＋DVT（易血栓性）
  • 血管炎の合併
  • 睡眠時無呼吸症候群の合併

⑨ 治療の基本原則

治療の3本柱

  • ① 原疾患の治療
  • ② 塩分制限（Na 6 g/日以内）
  • ③ 利尿薬の組み合わせ

利尿薬NGのケース

  • 血管透過性亢進
  • リンパ浮腫
  • DVT

※逆に病態悪化の危険

ループ利尿薬のコツ

  • 1日2回投与が原則
  • 慢性使用ではブレーキ現象
  • 低Alb血症では効果減弱
  • NSAIDs併用で効果減弱

抵抗性浮腫への段階療法

  • ループ倍量化（尿Na<50で）
  • サイアザイド系追加
  • MRA追加（効果1〜2日）
  • フロセミド持続静注
⑩ 診断アルゴリズム（AAFP 2022）
片側性 → 急性/亜急性 vs 慢性で分岐
経過ステップ内容
急性/亜急性1. Wells スコア高/中リスク → D-dimer ± 下肢エコー
2. DVT除外後蜂窩織炎・外傷・Baker嚢胞破裂・コンパートメント症候群を考慮
3. 左下肢May-Thurner症候群 → MR venography
慢性Duplex超音波（逆流あり）→ CVI
Duplex超音波（逆流なし）→ リンパ浮腫・リンパシンチグラフィを検討
両側性 → 薬剤 → SAS → 全身性疾患の順で評価
ステップYesNo → 次へ
1. 原因薬剤あり？薬剤中止・変更↓
2. SASの徴候？STOP-Bang ≧ 3 → 睡眠検査↓
3. 全身性疾患の評価BNP・肝機能・TSH・BMP・尿蛋白/Cr比
4. 全身性疾患なしDuplex超音波 → CVI or リンパ浮腫 or その他
5. BNP上昇 or 肺水腫→ 心エコー
⑪ SORT Key Recommendations（エビデンスレベル）
推奨レベル根拠
下肢浮腫は片側/両側、急性/慢性で層別化するC臨床レビュー・専門家意見
初期検査：BNP、肝機能、TSH、BMP、尿蛋白/Cr比C臨床レビュー・専門家意見
Wells スコアは急性片側下肢浮腫のDVT診断に最も有用（LR+ = 5.2）ASR/メタ解析
STOP-Bangは慢性両側下肢浮腫でSASスクリーニングに使用C臨床決断ルール
慢性浮腫にはDuplex超音波（逆流評価）でCVI vs リンパ浮腫を鑑別C臨床レビュー・専門家意見
圧迫療法前にABIを測定しPADを除外するC臨床レビュー・専門家意見
利尿薬抵抗性CHFではトルセミド > フロセミドASR/メタ解析
圧迫療法はほとんどの下肢浮腫に有効C臨床レビュー・専門家意見
⑫ ピットフォール
最大の落とし穴：単一病態で説明しようとすること。浮腫は常に複合病態を想定する。

見落とし代表例

  • うっ滞性皮膚炎＋蜂窩織炎
  • ネフローゼ＋DVT
  • 血管炎の合併
  • 睡眠時無呼吸症候群の合併
  • 悪性腫瘍（上肢DVT・RS3PE・片側乳癌関連）
  • 非全身性浮腫への不適切な利尿薬使用
