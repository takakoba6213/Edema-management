### 構成（4タブ）

| タブ | 内容 |
|------|------|
| **🔍 鑑別アルゴリズム** | PODCAST法で臨床情報を入力 → Semantic Qualifier自動生成 → スコア順に鑑別候補を表示 |
| **📋 チェックリスト** | 12セクションの診療フロー（原則・病態生理・問診・診察・検査・胸腹水・上肢・下肢・全身・薬剤性・治療・ピットフォール） |
| **👁 一目まとめ** | 4機序・年齢別・緊急疾患・Wells基準・STOP-Bang・SORT推奨・診断アルゴリズムの網羅表 |
| **📚 参考文献** | DOIリンク付き文献一覧 |

### 主な機能

- **PODCAST鑑別アルゴリズム** — P/O/D/C/A/S/T の各項目を選択 → 50以上の鑑別疾患からマッチ度順に表示
- **Semantic Qualifier** — 選択内容から1行の臨床要約文を自動生成、コピー可能
- **Wells基準 / STOP-Bang** — DVTの検査前確率、SASスクリーニングスコア
- **身体所見の尤度比（LR+）** — S3（11.0）、腹部頸静脈逆流（6.4）、JVD（5.1）など
- **チェックリスト** — セッション中の進捗保持、全展開/全折畳/リセット
- **レスポンシブ** — PC・タブレット・スマートフォン対応
- **スタンドアロン** — 外部依存なしの単一HTMLファイル

### GitHub Pages 有効化手順

1. リポジトリの **Settings** を開く
2. 左メニュー **Pages** を選択
3. Source: `Deploy from a branch` → Branch: `main` / `/ (root)` → **Save**
4. 数分後に `https://takakoba6213.github.io/Edema-management/浮腫まとめ_復習用.html` でアクセス可能
